Taichi

Taichi is a framework capable of running Xposed modules that changes the behaviour of the Android system/apps. Taichi can run in either a rooted environment or non-rooted environment. Taichi is compatible with Android L ~ Android Q.

Download

Download the latest Taichi app Download the latest Magisk module

Features

  • Taichi fully supports Android 9 (A.K.A. Android P)

  • Taichi can run without root and TWRP

  • Taichi is not globally effective, it can only enable Xposed features for selected apps. The system runs the same as if there is no Xposed framework present.

    This means that Taichi can easily bypass SafetyNet and is fully compatible with many financial/mobile banking/weird apps.

  • In the majority of the cases, modules enabled in Taichi can take effect without rebooting your system

  • Taichi is more difficult to be detected, its low intrusiveness made Xposed modules work without modifying Android Runtime, and of course, it leaves no trace in the global systematic environment

    If needed, it can bypass various types of Xposed detection.

Usage

Taichi has two modes: Non-root mode and Magisk mode; if you can't or you don't want to root, you can use the non-root mode, if you wish for more control over the system, you may try the Magisk mode

Non-root mode

Taichi uses non-root mode by default, all you need is to install the app. If you want to use Xposed modules, follow these instructions:

    1. Click the FAB in the bottom right, and click Create App

    2. Choose the app you wish to use Xposed module on, for example, WeChat

    3. Click Create App in the bottom and wait patiently

    4. After app creation finish, Taichi will ask you to remove the original app. This is necessary only in non-root mode.

    5. Follow in-app instructions to finish remaining procedures, like install and optimisations

    6. Again click the FAB and choose Manage Modules

    7. In the module management activity choose your desired Xposed module, for example, MDWeChat. You may be warned for version compatibility issues, so remember to download a compatible version

    8. Force Close your chosen app, in this case, WeChat, and reopen the app. At this point, the Xposed module should be in effect. (No reboot needed)

Magisk mode

The non-root mode does not require root/unlocking BL, therefore it is indeed easy to use, however, there are flaws like unable to modify global system environment and must uninstall the original app

So we made a Magisk module, granting Taichi even more powerful features and breaking the various limitations that exist only in non-root mode

Taichi will automatically switch over to Magisk mode after you install the Magisk module. If you remove or disable the Magisk module at any point, Taichi will again automatically revert to non-root mode

Xposed module support

Taichi is a framework, so developers can make their own module to run in Taichi. Taichi is also compatible with most Xposed features with a minimal amount of exception, so if you can make Xposed flavored modules, it'll most likely work in Taichi
